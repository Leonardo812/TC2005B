Preguntas a Responder:

  • ¿Cuál es la diferencia entre Internet y la World Wide Web?

    • El internet se refiere a la inmensa red conectada de computadoras y World Wide Web es la coleccion de diversas paginas web que estan en esa red de computadoras

  • ¿Cuáles son las partes de un URL?

    • Protocolo https, Subdominio, Nombre de dominio,Extensión de dominio, Ruta, Parámetro, Etiqueta, Dos puntos, Doble barra invertida.

  • ¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?

    • Indican la acción que se desea realizar para un recurso determinado

  • ¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?

    • Se hace uso del método post. Este método sirve envia datos a un servidor y crear un recurso.

  • ¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?

    • El método de petición GET, recupera los datos del URL

  • Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?

    • No, pues la solicitud fue exitosa

  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?

    • No, pues los errores 4XX se presentan cuando un usuario realiza una solicitud erronea

  • ¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?

    • Este error nos indica que el problema es interno del servidor,por lo que el desarrollador deberá corregir lo relacionado al error.

  • ¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.

    • Los tags y atributos desaprobados son partes del estándar HTML no recomendado. Mayormente debido a que el estandar se ha ido actualizando agregando, removiendo o desaprovando caracteristicas del lenguaje con el pasar del tiempo. Ejemplos:

    • Strike
    • Font
    • Center
    • BaseFont

  • ¿Cuáles son las diferencias principales entre HTML 4 y HTML5?
    • Reduccion de etiquetas redundantes
    • Cambios de elementos y estilos

  • ¿Qué componentes de estructura y estilo tiene una tabla?

    • La etiqueta "table", nos permite definir tablas, pero para modificarlas se necesitan otras etiquetas que solo pueden ser usadas dentro de una etiqueta de "table"
