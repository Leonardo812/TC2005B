Preguntas

  1. Qué diferencias y semejanzas hay entre Java y JavaScript?

    2. Java es un lenguaje de programación diseñado para aplicaciones no considerados ordenadores, como las tablets, consolas de a bordo, celulares, etc. Esto debido a que cuentan con recursos limitados (memoria, disco duro). Java es un lenguaje que necesita compilar usando Java Virtual Machine permitiendo que la maquina lo interprete. Java cuenta un tipado fuerte y JavaScript por otra parte tiene un tipado débil.

    JavaScript es un lenguaje de programación web, usado comunmente para aplicaciones y páginas web. Es considerado más creativo, pues esta dedicado al desarrollo de aplicaciones denominadas front end.

  2. Qué métodos tiene el objeto Date? (Menciona al menos 5*)

    3. El objeto date trabaja con horas y fechas

    • getMinutes() ----> Devuelve los minutos actuales (0 a 59).
    • getHours() ----> Devuelve la hora actual (0hrs - 23 hrs)
    • getDay() ----> Devuelve el día de la semana (0=Domingo, 6=Sabado)
    • getDate() ----> Devuelve el día del mes, (1-31)
    • getMonth() ----> Devuelve el número de mes (0= Enero, 11= Diciembre)
    • getFullYear() ----> Devuelve el año en 4 dígitos. (XXXX)


  3. ¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

    4. En JavaScript [] representa un arreglo

    • push() ----> Inserta un elemento en el final de un arreglo.
    • pop() ----> Remueve un elemento en el final de un arreglo.
    • length ----> Mide el tamaño del arreglo.
    • shift() ----> Remueve un elemento en el principio del arreglo.
    • unshift() ----> Inserta un elemento en el principio del arreglo.
    • Array.isArray() ----> Determina si el valor dado es un arreglo.


  4. ¿Cómo se declara una variable con alcance local dentro de una función?

    5. " Let " Declara variables limitando su alcance a un bloque, declaración, o expresión en el lugar donde se usa.
